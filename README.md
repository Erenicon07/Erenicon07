### My Projects🧠📂
**Division Game**
https://github.com/Erenicon07/Division-Game-<br>

**Game Information🎮**<br>
This is a division game developed in Unity.
The main goal of the game is to achieve the highest possible score before the time runs out.

However, there are some challenges in the game. You have three chances to make mistakes — if you make a third mistake, the game is over.

Additionally, each correct answer adds 5 seconds to the timer, while each incorrect answer subtracts 5 seconds.
This mechanic encourages players to stay both fast and accurate.
